CUDAMatLib
==========
A library offering GPU parallel computation for array and matrix data types with CUDA.

Requirements
------------
* Linux/Unix
* g++ 4.2 or higher
* NVIDIA VGA card
* [NVIDIA CUDA toolkit](https://developer.nvidia.com/cuda-downloads) (>= CUDA 5.0)

Installation & Test
-------------------
* Execute "make" to create binaries in bin/ for testing purpose.
* Use the header files in include/ folder directly (refer to src/\*.cu for use cases).
